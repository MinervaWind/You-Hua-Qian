# Changelog

All notable changes to 账本 will be documented in this file.

Inspired by Minerva, coded by Claude.

## [v0.8.0] - Upcoming

### Added
- 长期项目追踪模块（统计&计划页）
  - 新建项目：名称、开始日期、颜色标记、预设已投入、备注
  - 归档功能：记录归档日期，展示始于/归档于/共xx天，可重新激活
  - 记账时可选归属项目，默认关闭
  - 自动统计：累计支出、已进行天数、记录笔数、涉及分类
- 分类大类支持前移/后移排序
- 分类大类支持自定义颜色，饼图同步渲染
- 理财计划支持编辑金额和备注

### Changed
- 饼图预设颜色池扩充至20个

---

## [v0.7.0] - 2026-06-10

### Added
- 编辑历史流水记录（支持修改金额、日期、账户、分类、备注，余额自动正确更新）
- 月份选择器新增"全部流水"选项，显示所有历史记录，默认展示20条
- 月历点击当天弹窗底部新增"在这天记一笔"，跳转记账页并预填日期
- 饼图支持收入来源/支出分布切换，默认显示支出
- 记账成功后金额框自动聚焦，方便连续记账
- 月份选择器右侧新增"今天"快捷按钮，一键跳回当月

### Changed
- 所有删除确认从浏览器原生 confirm() 改为页面内自定义弹窗

### Fixed
- 筛选器函数名不匹配导致账户筛选无效的问题（renderOverviewList → renderOverview）

## [v0.6.0] - 2026-06-10

### Added
- 最近记录默认折叠显示10条，可展开查看全部，右上角显示总条数
- 账户页顶部新增总资产和待付款合计汇总
- 常用快捷入口：填好表单点 ★ 存入，下次一键填入表单
- 待付款支持每月/每年重复，确认付款后自动生成下一期
- Excel 导出新增"分类汇总"第三个 sheet

### Changed
- 署名更新为 "Inspired by Minerva, coded by Claude"

## [v0.5.0] - 2026-06-10

### Added
- 待付款功能
  - 记录金额、付款日期、关联账户、分类
  - 逾期/7天内/正常三种状态颜色区分
  - 确认付款后自动生成支出记录并扣除账户余额
- 账户卡片显示该账户待付款合计（威慑力设计）
- 记账&概览页7天内待付款提示条，点击跳转统计页

### Fixed
- 默认主题补全暗色模式（`@media prefers-color-scheme: dark`）
- 切换主题时默认主题改为移除 data-theme 属性而非设为空字符串

## [v0.4.0] - 2026-06-10

### Added
- 完整备份/恢复功能：导出 JSON 包含全部数据（账户、分类、记录、计划、待付款）
- FILE_ID 写死在文件里的存储隔离方案：每个文件有独立标识，互不干扰
- 首次命名流程：首次打开弹出命名提示，确认后生成写入 FILE_ID 的新文件供保存
- 导入 JSON 备份时自动恢复所有数据

### Changed
- 存储 key 从固定的 `ledger_pro_v1` 改为 `ledger_[FILE_ID]` 动态方案
- 导出页新增"完整备份/恢复"区块，与流水导出分开

## [v0.2.0] - 2026-06-10

### Added
- 四选项卡结构：记账&概览 / 账户&分类 / 统计&计划 / 导出&设置
- 多账户管理：添加/编辑/删除账户，颜色标记，余额校正，账户间转账
- 二级分类（分类+细则），支出/收入分开管理，可自由增删改名
- 三套主题：默认（跟随系统）/ Rosé Pine（Dawn/Moon）/ Cyan/Grey
- 三种理财计划：余额预警、月度限额、年度收入预期，进度条实时展示
- 月历视图：点击日期弹出当天明细
- 支出分布饼图（Chart.js）
- 月度趋势折线图
- Excel 导出（流水明细 + 月度汇总两个 sheet）
- CSV 导出
- 账本标题可自定义
- Ctrl+Enter 快速提交记账

### Changed
- 从单页面原型完整重写
