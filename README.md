# 账本 · You-Hua-Qian

> 基于二次元同人女小范围群体画像自研的高自由度多元记账本。

一个单文件 HTML 本地记账工具。不需要安装，不需要注册，不需要联网，双击打开即用。数据不经过任何服务器。

账本名可理解为“有”花钱、“又”花钱和“you”花钱等等……

---

## 功能一览

**记账**
- 支出 / 收入分类记录，支持二级分类（分类 + 细则），可自由增删改名
- 多账户管理，账户间转账，余额校正
- 常用快捷入口，填好表单一键存入，下次点击自动填入
- 支持编辑历史记录（不鼓励但支持日后补账和紧急假账）

**查看**
- 按月查看，或切换到「全部流水」浏览全部历史
- 月历视图，点击任意一天查看当日明细，或直接在那天新建记录
- 收入来源 / 支出分布饼图，月度趋势折线图

**计划**
- 三种理财计划：账户余额预警、月度支出限额、年度收入预期
- 待付款管理：记录未来需要支付的款项（防尾火功能），支持每月 / 每年重复，到期提醒
- 长期项目追踪：记录你为某件喜欢的事累计投入了多少（v0.8.0）

**其他**
- 可跟随系统日夜模式的三套配色主题：默认黑白、Rosé Pine、Cyan / Grey
- 完整备份 / 恢复：导出 JSON 文件，换电脑或分享给朋友时一键还原
- 导出 Excel（流水明细 + 月度汇总 + 分类汇总）或 CSV
- 多文件隔离：每个文件有独立存储标识，不同用户数据互不干扰

## 使用方法

1. 在 [Releases](../../releases) 页面下载最新版本的 `.html` 文件
2. 用浏览器打开（推荐 Chrome / Edge）
3. 首次打开时为你的账本起一个名字，浏览器会生成一个绑定了该名字的新文件
4. 保存这个新文件，之后**用新文件**记账即可

> 数据存储在浏览器的 localStorage 中，清除浏览器数据前请先导出完整备份。

## 分享给朋友

将原始未命名文件发给朋友，朋友首次打开时输入自己的名字，生成属于他们的专属版本，与你的数据完全隔离。
